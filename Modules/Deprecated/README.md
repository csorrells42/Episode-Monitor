# Deprecated Modules

This folder keeps retired implementation experiments available for reference without compiling them into Episode Monitor. `EpisodeMonitor.csproj` excludes `Modules\Deprecated\**\*.cs` from the active build.

Current archived material includes the old rolling face measurement-learning model, old measurement preview/package/plan classes, old vision evaluator/smoke/verifier tooling, and older reconstruction experiments. Keep active 3DDFA_V2 ONNX avatar capture, MediaPipe/OpenCV cue tracking, event evidence, and camera code under the main `Modules` branches.
